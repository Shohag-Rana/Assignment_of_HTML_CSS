

# A Travelling Responsive Webpage Design

In this project i am designing a responsive webpage using HTML, CSS. The webpage have 40% HTML code 50% CSS code and 10% are CSS framework Bootstrap code.


## Technologies

```bash
1. HTML 5
2. CSS 3
3. Bootstrap 4
4. Font Awesome
```
